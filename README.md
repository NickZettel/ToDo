# ToDo
Flask Todo app

A group project done for CST8319 at Algonquin College. 
Project Manager: Jeff J
Lead Developer: Nick Z
Frontend Developer: Emad S
Backend Developer: Xuemin Z


The project is a web app which allows a user to create an account, sign in, and view/add/delete/modify tasks. 
The app was made using Flask. The frontend uses Bootstrap.

Requires:
python (https://www.python.org/downloads/)
flask (pip install flask) 


Instructions:

Download zip file, extract all.
Enter root file where you can see py files and other files.
Click address bar in file explorer
type cmd + enter
type "flask --app app run --debug " without the quotations
hit enter
open a browser and go to http://127.0.0.1:5000/
