Instructions to run locally:

Requires: python and flask


Inside root folder containing all files, click address bar.

type cmd

hit enter

type: flask --app app run --debug

hit enter

Should see 
Running on http://127.0.0.1:5000

Open a web browser 
navigate to:
http://127.0.0.1:5000/
